## Codebase Patterns
- Tailwind CSS v4 with `@plugin` syntax for plugins (not `plugins: []` in config)
- Prose styling customization uses CSS custom properties (--tw-prose-*) in `@layer base`
- Font families: Raleway (sans), Roboto Serif (serif), Oxygen Mono (mono), Oswald (display headings)
- Color tokens use CSS custom properties in :root and .dark
- Build requires DATABASE_URL, NEXTAUTH_SECRET, NEXTAUTH_URL, DISCORD_CLIENT_ID, DISCORD_CLIENT_SECRET, TWITCH_CLIENT_ID, TWITCH_CLIENT_SECRET env vars
- Worktree builds will fail at prerender stage without a real database — typecheck (`npx tsc --noEmit`) is the reliable quality gate
- The `@eslint/eslintrc` package may need to be installed as devDep for type checking to pass in worktrees
